# Reference

This folder is the internal encyclopedia of Maana.

It is different from research.

- Research asks what other products or institutions do.
- Reference defines our own canonical articles for symbols, concepts, motifs, and related topics.

These documents should eventually become reusable editorial assets for:

- poem pages
- symbol pages
- search explanations
- recommendations
- editorial workflows
- future AI grounding

## Article Rule

Reference articles should be stable, citation-friendly, and organized around canonical knowledge
rather than temporary product decisions.
