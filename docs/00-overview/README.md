# README

Maana is a knowledge-first digital library for exploring poetry through
human experience, symbolism, philosophy, and literary relationships.

It is not just a collection of poems. It is a structured map of how poets, poems,
themes, symbols, languages, and civilizations connect.

## Why The Name Fits

`Maana` is derived from `ma'na` (معنى), meaning significance or what lies behind words.
That is exactly the transition the platform is designed to support:

- from text to understanding
- from verse to interpretation
- from language to insight

## What This Documentation Does

- Defines the intellectual model of the product.
- Clarifies product goals before code is written.
- Describes how discovery, explanation, and interpretation should work.
- Gives future contributors a shared vocabulary.

## What Comes First

The first implementation target is not a screen. It is a set of clear definitions.

Before designing pages, we need to answer:

- What is a poem?
- What is a verse?
- What is a theme?
- What is a symbol?
- What is a human experience?
- How do translations relate to originals?
- How do contradictory meanings coexist?
