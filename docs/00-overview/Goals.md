# Goals

## Product Goals

- Help readers discover poetry through themes, symbols, and human experiences.
- Help readers cultivate the inner vocabulary needed to understand difficult poetry.
- Build a reusable domain model that supports web, mobile, CMS, and AI features.
- Create durable editorial standards for commentary and interpretation.
- Make literary relationships searchable and explorable.
- Support serious reading while remaining accessible to new readers.

## MVP Goals

- Define the core entities: poet, poem, verse, theme, symbol, language, translation.
- Ship strong poem and symbol exploration paths.
- Introduce foundations and readiness as first-class reading aids.
- Support search by text, poet, theme, symbol, and language.
- Build a structured content model for commentary and references.
- Document the ontology clearly enough that implementation choices become obvious.
