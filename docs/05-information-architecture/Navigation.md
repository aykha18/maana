# Navigation

## Primary Navigation

- Home
- Explore
- Academy
- Search
- My Map

## Secondary Navigation

Within poem pages:

- Poem
- Foundations
- Commentary
- Related
- Reflection

Within concept or experience pages:

- Definition
- Why It Matters
- Examples
- Related Poems
- Related Lessons

## Navigation Principles

- Reading must remain the central action.
- Foundations should be one click away from a poem, never buried.
- Academy content should feel like preparation for reading, not a detached learning portal.
- My Map should be discoverable but never dominate the reading interface.

## Contextual Entry Points

- poem -> foundation
- foundation -> lesson
- lesson -> related poem
- reflection -> recommended next reading
- my map -> readiness-based recommendation
