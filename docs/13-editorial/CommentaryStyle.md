# CommentaryStyle

Commentary should help the reader understand a poem without pretending that every reading
is equally strong.

## Purpose

Commentary explains text, context, symbols, themes, and interpretive possibilities.
It should support reading, not overwhelm it.

## Style Rules

- Start from the text before moving to interpretation.
- Distinguish observation from inference.
- Name ambiguity when it matters.
- Avoid false certainty.
- Prefer clarity over ornamental prose.
- Preserve cultural and historical nuance.

## Structure

Good commentary usually covers:

- what the line or poem says
- what the key symbols suggest
- which themes are present
- what historical or literary context matters
- where multiple readings are possible

## Canonical Rule

Commentary should align with the ontology.
If a concept, theme, symbol, or human experience is central to the reading,
use the canonical terms defined in `04-ontology/CanonicalOntology.md`.
