# CitationRules

Citation rules define how the platform attributes sources behind texts, translations,
commentary, editorial claims, and media.

## Purpose

The platform should be able to answer:

- Where did this text come from?
- Who made this translation?
- Which commentary tradition supports this claim?
- Which scholarly source is being referenced?

## Citation Types

The system should support citations for:

- critical editions
- manuscript sources when relevant
- public domain translations
- modern translations
- scholarly books
- journal articles
- encyclopedia or reference works
- audio recitations

## Minimum Fields

Each citation should be able to store:

- source type
- author or editor
- title
- edition or version
- publication year when known
- page, section, or line reference when possible
- URL or archive location when applicable

## Rule

A poem may contain multiple citation layers:

- source-text citation
- translation citation
- commentary citation
- evidence citation
- reference article citation

## Editorial Standard

If a claim cannot be traced to a direct source, it should either:

- be marked as editorial synthesis, or
- be revised until its basis is clearer.
