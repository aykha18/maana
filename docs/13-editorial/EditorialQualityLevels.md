# EditorialQualityLevels

Quality levels describe how trustworthy, polished, and interconnected a poem's annotations are.

Unlike coverage levels, quality levels measure editorial maturity rather than just how many sections exist.

## Bronze

Basic reading support.

- original text verified
- basic translation present
- minimum metadata complete

## Silver

Usable learning support.

- word annotations for key lines
- core ontology tags applied
- basic commentary checked for clarity

## Gold

Strong interpretive support.

- historical context
- thematic analysis
- symbol notes where relevant
- claims are evidence-aware

## Platinum

Deep scholarly-style reading support.

- mystical, philosophical, or literary commentary where relevant
- multiple interpretive layers handled carefully
- stronger internal consistency across notes

## Diamond

Fully interconnected editorial asset.

- strong citations
- strong evidence classification
- cross-linked ontology and reference articles
- relationship mapping to poems, poets, themes, and symbols
- suitable as a benchmark model for future poems

## Product Use

These levels can be shown to users and editors.

Examples:

- This poem has Gold-level annotations.
- This poem has Diamond-level annotations.
