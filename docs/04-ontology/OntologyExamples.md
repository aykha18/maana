# OntologyExamples

This document shows worked examples of how the ontology should be used.

Abstract definitions are necessary, but examples make the model legible to future editors,
designers, and engineers.

## Example 1

### Question

What is love?

### Human Experiences

- longing
- hope
- loss

### Themes

- separation
- devotion
- remembered love

### Symbols

- night
- rose
- mirror

### Concepts

- absence
- desire
- memory

### Commentary Use

Commentary can explain how the poem moves from felt longing to symbolic expression
without collapsing romantic and mystical readings into one.

## Example 4

### Poet-Centered Mapping

Ghalib

- longing

### Themes

- separation
- mortality
- faith

### Symbols

- wine
- dust
- mirror

### Literary Devices

- irony
- hyperbole
