# MVP

## Objective

The MVP should prove that Maana can do two things at once:

1. Help readers discover important poems.
2. Help readers become more ready to understand them.

## MVP Scope

### Reading Core

- poem page
- poet page
- theme page
- symbol page
- experience page
- search

### Reader Formation Core

- foundations for understanding block on poem pages
- concept and experience explainer pages
- short academy lessons
- reflection prompts
- lightweight readiness status

## Required MVP Behaviors

### Foundations Block

Each selected poem can show:

- 3-5 relevant foundations
- why each foundation matters
- a path to learn the foundation before reading

### Lesson Format

Each lesson should support:

- one core question
- one short explanation
- examples from poetry or culture
- one reflection prompt
- links to related poems

### Readiness Output

The MVP should support simple, humane states such as:

- new to this idea
- lightly explored
- well prepared

### Reflection

Reflection should be available on:

- selected lessons
- selected poem pages

## Content Targets

- 10-20 foundation concepts or experiences
- 10 short lessons
- 25-50 poems with explicit foundations
- 5-10 reflection prompts reused across poems and lessons

## MVP Success Condition

The MVP succeeds if a new reader can:

1. land on a difficult poem
2. understand what background would help
3. learn one prerequisite quickly
4. return to the poem
5. feel meaningfully more prepared to read it

## Explicit Non-Goals

- full course platform
- gamified streak-heavy learning
- quizzes as the primary learning mechanic
- AI-generated interpretation as the default path
