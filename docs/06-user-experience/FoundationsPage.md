# FoundationsPage

## Purpose

This page explains one prerequisite idea that helps readers approach poems more deeply.

Examples:

- longing
- impermanence
- alienation
- Sufi symbolism
- modernity

## Page Structure

### 1. Definition

Short, reader-facing definition.

### 2. Why It Matters

Explain how the idea changes reading.

### 3. Lived Examples

Use concrete human examples before abstract analysis.

### 4. Cross-Disciplinary Links

Possible sections:

- poets
- films
- music
- psychology
- religion
- philosophy

### 5. Related Poems

Show poems where this foundation is central.

### 6. Related Lessons

Link to short academy lessons for deeper preparation.

### 7. Reflection Prompt

One optional question connecting the idea to lived experience.

## UX Rules

- This page should feel richer than a glossary entry but lighter than an essay.
- The user should be able to move from definition to poem quickly.
