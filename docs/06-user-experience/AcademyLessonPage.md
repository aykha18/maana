# AcademyLessonPage

## Purpose

The lesson page is a short learning unit that builds familiarity with one human
experience, concept, or question.

## Target Shape

- 3-7 minutes
- one clear question
- one clear takeaway
- one path back into reading

## Page Structure

### 1. Lesson Question

Examples:

- What is longing?
- Why do humans grieve?
- What is transcendence?

### 2. Short Explanation

Concise conceptual framing in plain language.

### 3. Multiple Lenses

Small sections drawn from:

- story
- poetry
- philosophy
- psychology
- neuroscience
- religion or civilization

### 4. One Reflection Prompt

Reflection should be open-ended and experiential.

### 5. Related Poems

End with poems that become more legible after the lesson.

### 6. Readiness Effect

The lesson should update the user's familiarity with the topic without presenting
that change as a test score.

## UX Rules

- No quizzes by default.
- No course-like intimidation.
- The lesson should always point back to literature.
