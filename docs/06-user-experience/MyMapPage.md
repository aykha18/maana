# MyMapPage

## Purpose

My Map shows the user's exploration across human questions, experiences, concepts,
poems, and reflections.

## Core Sections

### 1. Readiness Overview

Show gentle states such as:

- new to this idea
- lightly explored
- well prepared

### 2. Explored Experiences

Examples:

- longing
- grief
- mortality

### 3. Explored Questions

Examples:

- What is love?
- Why do humans suffer?

### 4. Reflections

Private-by-default journal history or reflection count.

### 5. Suggested Next Steps

Recommendations based on what the user has explored but not yet connected.

## UX Rules

- The map should visualize exploration, not score identity.
- It should encourage return to poems and ideas, not endless dashboard checking.
